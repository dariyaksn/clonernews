# ClonerNews — Hacker News UI 🚀

**ClonerNews** — это современное веб-приложение для мониторинга технологических новостей, вакансий и обсуждений, созданное на базе официального Hacker News API.

## 🔗 Репозиторий проекта
[https://01.tomorrow-school.ai/git/tzhanel/clonernews](https://01.tomorrow-school.ai/git/tzhanel/clonernews)

## ✨ Основные цели и возможности (Objectives)

### 1. Обработка данных (Posts & Comments)
* **Разнообразие контента**: Поддержка типов `stories`, `jobs`, `polls` и полноценная работа с комментариями.
* **Сортировка**: Весь контент отображается в хронологическом порядке (от новых к старым).
* **Связи**: Строгое соблюдение иерархии "родитель — комментарий".
* **Вложенность (Optional)**: Реализована рекурсивная отрисовка вложенных веток комментариев.

### 2. Динамическая загрузка (Lazy Loading)
* Реализована порционная загрузка данных. Посты не загружаются все сразу, а подгружаются только по клику на кнопку "Load More", что экономит ресурсы API и трафик пользователя.

### 3. Живые данные (Live Data)
* **Авто-мониторинг**: Приложение проверяет наличие обновлений на сервере каждые **5 секунд**.
* **Система уведомлений**: При появлении новых постов в текущей категории, пользователь получает мгновенное всплывающее уведомление для обновления ленты.

### 4. Оптимизация и защита от Rate Limit
Для предотвращения блокировок и избыточной нагрузки на API реализованы:
* **Throttling**: Ограничение частоты нажатий на кнопки управления.
* **Debouncing**: Задержка обработки событий при переключении фильтров.
* **Batch Loading**: Использование `Promise.all` для параллельной загрузки деталей группы постов.
* **Retry Logic**: Автоматические повторные запросы при временных сбоях сети.

## 🛠 Технологический стек
* **HTML5**: Семантическая структура.
* **CSS3**: Адаптивный интерфейс с использованием переменных и анимаций.
* **Vanilla JavaScript (ES6)**: Объектно-ориентированный подход (классы) без использования сторонних фреймворков.
* **HackerNews Firebase API**: Основной источник данных.
* **Algolia API**: Оптимизированная выборка для раздела Polls.

## 🚀 Как запустить
1. Откройте VS Code.
2. Запустите файл `index.html` (рекомендуется использовать расширение *Live Server*).
3. Используйте фильтры для переключения категорий новостей.
